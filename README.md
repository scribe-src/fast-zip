### fast-zip

`fast-zip` implements a small Objective-C wrapper around `minizip` that allows for in-memory ZIP access. No ARC support at this time.

#### Setup

    $ git submodule update --init --recursive

#### Build

    $ make all

#### Run tests

    $ make test
